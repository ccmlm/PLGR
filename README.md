# plgr

## Usage

```
plgr 0.1.0

USAGE:
    plgr-distr [OPTIONS] --entries-path <ENTRIES_PATH> --privkey-path <PRIVKEY_PATH>

OPTIONS:
    -e, --entries-path <ENTRIES_PATH>
    -h, --help                           Print help information
    -p, --privkey-path <PRIVKEY_PATH>
    -u, --use-testnet
    -V, --version                        Print version information
```
